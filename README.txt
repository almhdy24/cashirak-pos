CASHIRAK POS - دليل التشغيل على ويندوز
========================================

1. قم بتحميل نسخة PHP لويندوز من الرابط:
   https://windows.php.net/download#php-8.4
   (اختر الإصدار "Non Thread Safe" بصيغة Zip)

2. قم بفك ضغط الملف داخل المجلد "php" الموجود بجانب هذا الملف.

3. شغّل ملف "start.bat" لبدء الخادم.

4. افتح المتصفح واذهب إلى:
   http://localhost:8000/install.php (أول مرة فقط)
   ثم اتبع التعليمات.

5. بعد التثبيت، استخدم:
   http://localhost:8000/login.php
   بيانات الدخول الافتراضية: admin / admin123

ملاحظة: بعد التثبيت، يُفضل حذف ملف public/install.php لأمان أكبر.
